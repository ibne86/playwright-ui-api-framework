# playwright-ui-api-framework

A Playwright (JavaScript) automation framework:
- **UI automation** (SauceDemo) using **Page Object Model (POM)**
- **API automation** (Reqres) using Playwright's built-in **APIRequestContext**
- **Multi-project setup**: UI runs on **Chromium/Firefox/WebKit**, API runs as a separate project
- **CI-ready** structure (GitHub Actions will be added)

## Tech Stack
- Playwright Test (JavaScript)
- POM for UI tests
- API tests with `request`

## Targets
- UI: https://www.saucedemo.com/
- API: https://reqres.in/
